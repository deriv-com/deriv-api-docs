### සංකේත API ඇමතුම සඳහා ගිවිසුම් යනු කුමක්ද?

ලබා දී ඇති සංකේතයක් සඳහා, දැනට පවතින ගිවිසුම් ලැයිස්තුව සහ එක් එක් ගිවිසුම සඳහා ඇති නවතම බාධක සහ කාල සීමාවන් ලබා ගන්න.
