### ගෙවීම් ක්‍රම API ඇමතුම යනු කුමක්ද?

දී ඇති රට සඳහා ලබා ගත හැකි ගෙවීම් ක්‍රම ලැයිස්තුවක් ආපසු ලබා දෙනු ඇත. ඉල්ලීම සත්‍යාපනය කර ඇත්නම් සේවාදායකයාගේ පදිංචි රට භාවිතා වේ.
