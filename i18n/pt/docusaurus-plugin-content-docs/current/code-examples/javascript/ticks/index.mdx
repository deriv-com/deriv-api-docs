<p>Vamos começar com o exemplo dos ticks porque é um bloco de código bastante simples de implementar. Não é necessário iniciar sessão na conta Deriv e autenticar a aplicação para experimentar.</p>
</RenderOfficialContents>

<RenderOfficialContents>
<p>O tick é uma forma de medir o movimento mínimo (menor variação) ascendente ou descendente no preço de uma negociação de matérias-primas. Subscrevemos à funcionalidade de ticks através do websocket da Deriv API, que irá fornecer novos dados de ticks a cada segundo. Por exemplo, pode criar um gráfico interativo de ticks com estes dados.</p>
</RenderOfficialContents>
