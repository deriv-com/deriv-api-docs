এই উদাহরণ একটি চুক্তি প্রস্তাব পাওয়ার জন্য। আপনি আপনার চুক্তির জন্য মূল্য, পেআউট এবং স্পট মান পেতে সক্ষম হবেন। নিষ্ক্রিয়তার টাইমআউটের ক্ষেত্রে এই সংযোগটিকে জীবিত রাখতে, Keep alive এর উদাহরণটি দেখুন।
