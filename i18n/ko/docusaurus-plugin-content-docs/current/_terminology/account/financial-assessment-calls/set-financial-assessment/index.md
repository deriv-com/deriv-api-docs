### 재무 평가 설정 API 호출이란 무엇인가요?

이 통화는 고객이 바이너리 옵션 거래와 관련된 위험을 이해할 수 있는 경험과 지식을 보유하고 있는지 분석하기 위해 고객의 답변을 기반으로 재무 평가 세부 정보를 설정합니다.
