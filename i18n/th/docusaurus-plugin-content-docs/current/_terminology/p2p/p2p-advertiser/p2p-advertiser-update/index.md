### การเรียกใช้งาน API เพื่ออัปเดตผู้ลงโฆษณา P2P คืออะไร?

อัปเดตข้อมูลของผู้ลงโฆษณา P2P สำหรับบัญชีปัจจุบัน สามารถใช้ได้โดยผู้ลงโฆษณา P2P ที่ได้รับการอนุมัติแล้วเท่านั้น
