### P2P বিজ্ঞাপন তৈরি API কল কি?

একটি P2P (পিয়ার টু পিয়ার) বিজ্ঞাপন তৈরি করে। শুধুমাত্র একটি অনুমোদিত P2P বিজ্ঞাপনদাতা দ্বারা ব্যবহার করা যেতে পারে।
