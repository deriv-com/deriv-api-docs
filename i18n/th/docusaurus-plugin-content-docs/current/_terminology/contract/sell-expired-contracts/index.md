### การเรียกใช้ API เพื่อขายสัญญาหมดอายุ คืออะไร?

การเรียกใช้งานนี้จะพยายามขายสัญญาที่หมดอายุและแสดงจำนวนสัญญาที่ถูกขายไปได้
