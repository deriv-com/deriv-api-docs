### 什麼是登陸公司詳細資料 API 呼叫？

該公司在不同的司法管轄區擁有多家獲得執照的子公司，這些子公司被稱為登陸公司（並且是 Deriv Group 的全資子公司）。

此呼叫提供每個登陸公司的資料。
