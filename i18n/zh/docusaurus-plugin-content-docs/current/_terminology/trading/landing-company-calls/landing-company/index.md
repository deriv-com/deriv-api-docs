### 什麼是登陸公司 API 呼叫？

該公司在不同的司法管轄區擁有多家獲得執照的子公司，這些子公司被稱為登陸公司。

此呼叫將為指定國家/地區的客戶傳回適當的登陸公司。

綜合指數和金融合約（外匯、股票指數、大宗商品）的登陸公司可能有所不同。
