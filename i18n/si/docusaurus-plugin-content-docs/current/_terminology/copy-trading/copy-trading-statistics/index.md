### පිටපත් ගනුදෙනු යනු කුමක්ද: සංඛ්‍යාලේඛන API ඇමතුම?

ගනුදෙනුකරුගේ කාර්ය සාධනය, ගනුදෙනු, අවදානම් සහ පිටපත් කරන්නන්ගේ සංඛ්‍යාලේඛන ලබා ගන්න.
