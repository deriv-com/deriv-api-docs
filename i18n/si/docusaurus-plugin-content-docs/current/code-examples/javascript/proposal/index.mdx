මෙම උදාහරණය ගිවිසුම් යෝජනාවක් ලබා ගැනීම සඳහා ය. ඔබට ඔබේ ගිවිසුමේ මිල, ගෙවීම සහ ස්ථාන වටිනාකම ලබා ගැනීමට හැකි වනු ඇත. අක්‍රිය කාල සීමාවලදී මෙම සම්බන්ධතාවය සජීවීව තබා ගැනීමට, සජීවීව තබා ගැනීම සඳහා උදාහරණය බලන්න.
