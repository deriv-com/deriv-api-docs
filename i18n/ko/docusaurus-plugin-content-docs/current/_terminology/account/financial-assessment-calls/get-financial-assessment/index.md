### 재무 평가 받기 API 호출이란 무엇인가요?

이 통화는 재무 평가 세부 정보를 가져옵니다.

'재무 평가'는 특정 랜딩 컴퍼니의 고객이 규제 및 고객알기제도(KYC) 요건으로 인해 작성해야 하는 설문지입니다.
