### P2P දැන්වීම් නිර්මාණ API ඇමතුම යනු කුමක්ද?

එය P2P නිර්මාණය (Peer to Peer) දැන්වීමක්. භාවිත කළ හැක්කේ අනුමත P2P දැන්වීම්කරුවෙකුට පමණි.
