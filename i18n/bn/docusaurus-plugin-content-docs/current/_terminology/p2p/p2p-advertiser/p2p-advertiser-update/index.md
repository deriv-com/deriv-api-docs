### P2P বিজ্ঞাপনদাতা আপডেট API কল কি?

বর্তমান অ্যাকাউন্টের জন্য P2P বিজ্ঞাপনদাতার তথ্য আপডেট করুন। শুধুমাত্র একটি অনুমোদিত P2P বিজ্ঞাপনদাতা দ্বারা ব্যবহার করা যেতে পারে।
