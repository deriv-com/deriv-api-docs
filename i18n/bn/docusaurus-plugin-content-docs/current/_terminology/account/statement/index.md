### স্টেটমেন্ট এপিআই কল কি?

প্রদত্ত অনুসন্ধানের মানদণ্ড অনুযায়ী অ্যাকাউন্ট লেনদেনের একটি সারসংক্ষেপ উদ্ধার করুন।
