### การเรียกใช้ API เพื่อตัวแทนการชำระเงิน: โอน คืออะไร?

การโอนผ่านตัวแทนการชำระเงิน - การเรียกใช้งานนี้จะมีให้ใช้ได้เฉพาะบัญชีที่ได้รับการอนุมัติเป็นตัวแทนการชำระเงินเท่านั้น
