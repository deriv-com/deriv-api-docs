### প্রতীক API কল জন্য চুক্তি কি?

একটি প্রদত্ত প্রতীকের জন্য, বর্তমানে উপলব্ধ চুক্তিগুলির তালিকা এবং প্রতিটি চুক্তির জন্য সর্বশেষ বাধা এবং সময়কাল সীমা পান।
