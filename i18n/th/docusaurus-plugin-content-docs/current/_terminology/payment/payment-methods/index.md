### การเรียกใช้งาน API เพื่อวิธีการชำระเงิน คืออะไร?

จะแสดงลิสต์รายการวิธีการชำระเงินที่มีให้ใช้ได้สำหรับประเทศที่ระบุ หากคำร้องขอได้รับการรับรองความถูกต้อง ประเทศที่พำนักของลูกค้าจะถูกนำมาใช้
