### Co to jest umowa kupna dla wielu kont API Call?

Kup umowę na wiele kont określonych przez parametr „tokeny”.

Uwaga, chociaż jest to autoryzowane połączenie, umowa nie jest kupowana dla autoryzowanego konta.
