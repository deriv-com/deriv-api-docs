### P2P বিজ্ঞাপনদাতা বিজ্ঞাপন API কল কি?

অনুমোদিত ক্লায়েন্ট দ্বারা তৈরি সমস্ত P2P বিজ্ঞাপন ফেরত দেয়। শুধুমাত্র একটি নিবন্ধিত P2P বিজ্ঞাপনদাতা দ্বারা ব্যবহার করা যেতে পারে।
