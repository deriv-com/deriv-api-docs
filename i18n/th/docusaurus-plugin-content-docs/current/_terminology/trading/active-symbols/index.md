### การเรียกใช้งาน API เพื่อสัญลักษณ์ที่ใช้งานอยู่ คืออะไร?

เรียกดูลิสต์รายการสัญลักษณ์ที่ใช้งานอยู่ในปัจจุบันทั้งหมด (ตลาดสินทรัพย์อ้างอิงที่มีสัญญาให้ได้ทำการเทรดหรือซื้อขาย)
