### การเรียกใช้งาน API เพื่อพอร์ตโฟลิโอ คืออะไร?

รับข้อมูลเกี่ยวกับพอร์ตโฟลิโอปัจจุบันของฉันเกี่ยวกับตราสารสิทธิที่โดดเด่น
