### การเรียกใช้ API เพื่อสัญญาสำหรับสัญลักษณ์ คืออะไร?

สำหรับสัญลักษณ์ที่กำหนด รับลิสต์รายชื่อของสัญญาที่มีอยู่ในปัจจุบัน รวมถึงขีดจำกัดเส้นระดับราคาเป้าหมายล่าสุดและระยะเวลาของแต่ละสัญญา
