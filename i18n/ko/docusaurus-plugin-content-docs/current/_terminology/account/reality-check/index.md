### 현실 확인 API 호출이란 무엇인가요?

고객의 거래 요약과 현실 확인 기능을 검색합니다.

'현실 확인'이란 세션 시작 후 경과된 시간과 관련 고객 수익/손실을 표시하는 것을 의미합니다.

현실 확인 기능은 특정 랜딩업체에 대한 규제 요건입니다.
