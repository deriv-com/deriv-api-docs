  <p>Ein Tick ist ein Maß für eine minimale Aufwärts- oder Abwärtsbewegung des Preises eines Handelsguts. Wir abonnieren die Tick-Funktionalität vom Deriv API Websocket, der jede Sekunde neue Tick-Daten zurückgibt. Sie könnten zum Beispiel ein interaktives Tick-Diagramm mit diesen Daten erstellen.</p>
